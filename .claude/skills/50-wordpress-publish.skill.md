# Skill Checklist Asset — Chuẩn Bị Đăng TikTok

## Checklist trước khi đăng

- [ ] Đủ 5–10 file ảnh theo thứ tự slide
- [ ] Tên file: rõ thứ tự (slide-01.png, slide-02.png...)
- [ ] Mỗi ảnh đúng tỉ lệ 9:16
- [ ] Text trên ảnh đọc được trên di động (font ≥ 36px)
- [ ] Đã chọn nhạc trend phù hợp tone bài
- [ ] Caption đã viết xong, có CTA + hashtag
- [ ] Script reply comment đã có sẵn
- [ ] Kenny đã review nội dung trước khi đăng

## Thư mục output

- Ảnh slide: `assets/tiktok-images/[YYYY-MM-DD-chu-de]/`
- Caption + hashtag: `outputs/[timestamp]/caption.txt`
- Script reply: `outputs/[timestamp]/reply-script.md`
- Report: `outputs/[timestamp]/report.json`

## Khung giờ đăng tốt

- Sáng: 7:00 – 8:00
- Tối: 20:30 – 21:30
