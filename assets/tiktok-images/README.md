# Ảnh Đầu Vào Cho TikTok

- Đặt ảnh nguồn vào thư mục này (`.jpg`, `.jpeg`, `.png`, `.webp`).
- Ưu tiên ảnh thật: campus, ký túc xá, lớp học, đời sống sinh viên, thành phố tại Philippines.
- Nên đặt tên file rõ nội dung, ví dụ: `cebu-campus-01.jpg`, `baguio-dorm-02.jpg`.
- Script `run_tiktok_leadgen.py` sẽ lấy vòng tua các ảnh này để tạo asset theo lịch đăng.
